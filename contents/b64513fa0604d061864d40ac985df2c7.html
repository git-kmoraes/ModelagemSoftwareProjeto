Ordem de venda

UMLUseCase
Untitled :: Model1 :: Ordem de venda

Description

  1. Usuário decide qual ação vender
  2. Usuário decide a quantidade a ser vendida (mercado fracionário ou lote)
  3. Usuário faz uma OV na corretora
  4. Usuário envia informações e o tipo de ordem (com ou sem gatilho)
  5. Corretora envia OV para B3
  6. Corretora retorna a execução da ordem
  7. Usuário visualiza o status da OV
  8. Usuário se quiser pode modificar a OV

8.1 Usuário consulta o histórico de OV 8.2 Usuário cancela a OV 8.3 Corretora cobra uma taxa e devolve o valor estabelecido pelo usuário
